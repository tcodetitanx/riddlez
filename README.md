## The Seven Puzzle Archetypes

| Archetype | Core Mechanism | Classic Example |
|-----------|----------------|-----------------|
| Constraint Satisfaction | Find unique arrangement fitting all rules | Einstein's Riddle |
| Game Theory | Reason about rational agents | Pirate Gold |
| Common Knowledge | Public announcements enable coordination | Blue/Green Eyes |
| State Space Search | Navigate from A to B avoiding forbidden states | Wolf, Goat, Cabbage |
| Encoding/Parity | Information hidden in unexpected signals | Prisoner Hats |
| Bayesian Reasoning | Update probabilities on new information | Monty Hall |
| Self-Reference | Truth values that reference themselves | Knights & Knaves |
