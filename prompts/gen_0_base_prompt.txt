RIDDLE GENERATION PROMPT - GENERATION 0 (BASE)
==============================================

You are a master puzzle designer. Create an original logic puzzle following these principles:

CORE REQUIREMENTS:
1. The puzzle must have a UNIQUE, VERIFIABLE solution
2. Setup should be explainable in under 100 words
3. Solution should require 3-5 logical steps
4. The "aha moment" should feel surprising yet inevitable

ARCHETYPE TOOLKIT (combine 2+ for novelty):
- Constraint Satisfaction: Find arrangement meeting all rules
- Game Theory: Reason about rational agents' behavior
- Common Knowledge: What everyone knows everyone knows
- State Space: Navigate from A to B avoiding forbidden states
- Encoding/Parity: Information hidden in unexpected places
- Bayesian: Update beliefs based on new evidence
- Self-Reference: Statements about statements

QUALITY CHECKLIST:
[ ] Is the solution unique? (prove alternatives fail)
[ ] Are constraints exactly tight? (not too loose, not impossible)
[ ] Does the answer surprise? (not obvious from setup)
[ ] Is it inevitable in hindsight? ("of course it must be that!")
[ ] Can every element be removed without breaking it? (if yes, remove it)

OUTPUT FORMAT:
1. SETUP: The puzzle scenario (under 100 words)
2. THE QUESTION: What must be solved
3. HINT: One subtle pointer
4. SOLUTION: Complete step-by-step proof
5. VERIFICATION: Show why alternatives fail
6. ARCHETYPE FUSION: Which mechanisms combined and how

AVOID:
- Hand-waving in solutions ("through analysis...")
- Multiple valid solutions
- Requiring specialized knowledge
- Direct reskins of famous puzzles
- Complexity for complexity's sake
